Deck Forge is a clash royale deck builder. You have 1 of 2 options:

Option 1: Create the deck yourself
As the name suggests, you can create the deck to your own accord, they will also display if the cards you use are also being utilised by the top 100 decks

Option 2: Create Deck
Deck forge will pick one of the 100 top decks and add it into your deck


After picking one of the 2 options. You can then either:
- Delete the deck you made if you want to try something new (Plays an easter egg gif)
- Copy the Deck into clash royale itself(works on mobile, can also work if you copy the link from ur laptop to ur phone)

Instructions to run:

To open the web app, run the command 'npm run dev'

run the command 'npm install animejs' one more time just in case

Open http://localhost:3000 with your browser to see the result.
